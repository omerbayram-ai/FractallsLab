## 🎯 Vision

Fractals Lab is not just a website — it's a **digital laboratory** designed to make the most abstract concepts of mathematics tangible, visual, and intuitive. 

Born from the frustration of learning fractals through static textbooks and dense academic papers, this project transforms **linear algebra, complex analysis, and chaos theory** into hands-on experiments that anyone with a browser can explore.

> *"The goal is not to teach fractals — it's to let you experience them."*

---

## ✨ Features

### 🧮 Core Mathematical Engines

| Module | Description |
|--------|-------------|
| **Mandelbrot & Julia Sets** | Real-time rendering with escape-time algorithm, infinite zoom, and color mapping |
| **Iterated Function Systems (IFS)** | Barnsley Fern and custom transformations via 2×2 matrix playground |
| **L-System Compiler** | Write your own axioms and production rules — watch plants grow algorithmically |
| **Chaos Game** | 100,000-point animation revealing how randomness creates deterministic order (Sierpinski Triangle) |
| **Box-Counting Dimension Calculator** | Interactive proof that fractal dimension is a measure of how shapes fill space |
| **Newton Fractals** | Visualize root-finding in the complex plane |
| **Weierstrass Function** | Explore the "monster function" — continuous everywhere, differentiable nowhere |

### 🎨 Interactive Learning Tools

- **Matrix Playground** — Manipulate 2×2 transformation matrices and see their effect on fractals in real-time
- **Coastline Paradox Simulator** — Measure the coastlines of Turkey, UK, Norway with different ruler sizes
- **Fractal Audio Lab** — Sonification of fractal structures into sound frequencies (rare feature!)
- **Historical Timeline** — From Euclid's smooth world to Mandelbrot's rough nature

### 📚 Educational Content

- Biographical deep-dive into Benoît Mandelbrot
- Step-by-step mathematical explanations
- Historical context and interdisciplinary connections (nature, art, physics)

### For Users
Simply visit the live site:  
👉 **[omerbayram-ai.github.io/FractallsLab](https://omerbayram-ai.github.io/FractallsLab/)**

No installation required. Works on any modern browser.

### For Developers
```bash

git clone https://github.com/omerbayram-ai/FractallsLab.git

# Navigate to the project
```
